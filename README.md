# Firefox Distraction Blocker

A personal Firefox extension that blocks distracting websites to help you stay focused.

## Features

- Blocks popular social media and entertainment sites
- Simple toggle to turn blocking on/off
- Persists your preference across browser sessions

## Blocked Sites

- Facebook
- Twitter / X
- Instagram
- TikTok
- Reddit
- YouTube
- Twitch
- Netflix
- Snapchat
- Pinterest
- Tumblr
- Discord

## Installation

1. Open Firefox and navigate to `about:debugging`
2. Click "This Firefox" in the sidebar
3. Click "Load Temporary Add-on..."
4. Select the `manifest.json` file from this folder

## Usage

Click the extension icon in the toolbar to see the status and toggle blocking on or off.
